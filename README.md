**Project Description**

This project focuses on analyzing sports-related data using Python to uncover patterns, perform statistical analysis, and build predictive machine learning models. The workflow includes data preprocessing, exploratory data analysis, feature engineering, clustering techniques, and predictive modeling.

The goal of the project is to demonstrate how data analytics and machine learning techniques can be applied to sports datasets to generate insights and predictions.

**Project Workflow**

The project follows a structured data science pipeline:

**1. Data Preprocessing**

Handling missing values

Data cleaning and transformation

Assigning appropriate data types

**2️. Exploratory Data Analysis (EDA)**

Descriptive statistics

Identifying trends and distributions

Correlation analysis

**3️. Data Visualization**

Visual insights are created using graphs and plots to understand the dataset better.

**Tools used:**

Matplotlib

Seaborn

**4️. Statistical Analysis**

Several statistical tests were applied to validate relationships in the data:

Shapiro Test (Normality test)

Levene Test (Variance equality)

Pearson Correlation

Other statistical measures

**5️. Feature Engineering**

Transforming raw data into meaningful features

Encoding categorical variables

Scaling numerical features

**6️. Clustering Analysis**

K-Means clustering was used to group similar data points and identify patterns within the sports dataset.

**7️. Machine Learning Model**

A Logistic Regression model was built to predict outcomes based on the dataset.

**Steps include:**

Train/Test split

Model training

Model evaluation using:

Accuracy Score

Confusion Matrix

**Technologies Used
Programming Language**

Python

**Libraries**

Pandas – Data manipulation

NumPy – Numerical computing

Matplotlib – Data visualization

Seaborn – Statistical visualization

Scikit-learn – Machine learning

SciPy – Statistical testing

Statsmodels – Statistical modeling

**Conclusion**

The analysis identified key trends and patterns in sports data, including player performance differences, correlations between statistics, and clusters of similar player profiles. It also highlighted important factors influencing performance outcomes through statistical analysis and machine learning models.

These insights can be presented to stakeholders using visualizations, summary metrics, and predictive results to support decisions in player evaluation, team strategy, talent identification, and performance improvement. Overall, the project enables data-driven decision-making in sports analytics.
